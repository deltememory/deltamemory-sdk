# DeltaMemory Documentation

Enterprise-grade documentation for DeltaMemory cognitive memory system.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm run start
```

## Structure

- `/pages` - Documentation pages (MDX)
- `/theme.config.tsx` - Nextra theme configuration
- `/public` - Static assets

## Writing Docs

Documentation is written in MDX (Markdown + JSX). See [Nextra docs](https://nextra.site) for features.
