# Quick Start

Get started with DeltaMemory in under 5 minutes.

## 1. Create Your Account

1. Go to [app.deltamemory.com](https://app.deltamemory.com)
2. Sign up with your email or OAuth provider
3. Create your first project

After signing up, you'll receive:
- Your unique API endpoint URL
- Your first API key

## 3. Basic Usage

<Tabs items={['TypeScript', 'Python']}>
  <Tab>
```typescript
import { DeltaMemory } from 'deltamemory';

const db = new DeltaMemory({
  apiKey: process.env.DELTAMEMORY_API_KEY,  // From dashboard
  baseUrl: process.env.DELTAMEMORY_URL,     // Your assigned endpoint
  defaultCollection: 'my-app'
});

// Ingest content with cognitive processing
const result = await db.ingest(
  'My name is Sarah. I work as a product manager at TechCorp and prefer async communication.',
  {
    datetime: new Date().toISOString(),
    speaker: 'user'
  }
);

console.log('Extracted facts:', result.facts);
// [
//   { fact: 'User name is Sarah', confidence: 0.95 },
//   { fact: 'Works as product manager', confidence: 0.92 },
//   { fact: 'Works at TechCorp', confidence: 0.90 },
//   { fact: 'Prefers async communication', confidence: 0.88 }
// ]

// Recall with structured context
const recall = await db.recall('What do I know about the user?');

console.log('Profiles:', recall.profiles);
// [
//   { topic: 'basic_info', sub_topic: 'name', content: 'Sarah' },
//   { topic: 'work', sub_topic: 'title', content: 'product manager' },
//   { topic: 'work', sub_topic: 'company', content: 'TechCorp' },
//   { topic: 'preference', sub_topic: 'communication', content: 'async' }
// ]

// Use pre-formatted context for LLM
console.log('Context for LLM:', recall.context);
```
  </Tab>
  <Tab>
```python
import asyncio
import os
from deltamemory import DeltaMemory, IngestOptions
from datetime import datetime

async def main():
    db = DeltaMemory(
        api_key=os.environ.get('DELTAMEMORY_API_KEY'),  # From dashboard
        base_url=os.environ.get('DELTAMEMORY_URL'),     # Your assigned endpoint
        default_collection='my-app'
    )

    async with db:
        # Ingest content with cognitive processing
        result = await db.ingest(
            'My name is Sarah. I work as a product manager at TechCorp and prefer async communication.',
            IngestOptions(
                datetime=datetime.now().isoformat(),
                speaker='user'
            )
        )

        print('Extracted facts:', result.facts)
        # [
        #   ExtractedFact(fact='User name is Sarah', confidence=0.95),
        #   ExtractedFact(fact='Works as product manager', confidence=0.92),
        #   ...
        # ]

        # Recall with structured context
        recall = await db.recall('What do I know about the user?')

        print('Profiles:', recall.profiles)
        # [
        #   UserProfile(topic='basic_info', sub_topic='name', content='Sarah'),
        #   UserProfile(topic='work', sub_topic='title', content='product manager'),
        #   ...
        # ]

        # Use pre-formatted context for LLM
        print('Context for LLM:', recall.context)

asyncio.run(main())
```

</Tabs>

## 4. Verify Connection

Check that your connection is working:

```bash
curl -H "X-API-Key: $DELTAMEMORY_API_KEY" $DELTAMEMORY_URL/v1/health
```

Expected response:

```json
{
  "healthy": true,
  "version": "0.1.0"
}
```
